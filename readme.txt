== Description ==

= Neeed, votre liste de shopping facile = 

Rassemblez toutes vos envies simplement grâce à Neeed et affichez vos envies sur votre blog dans vos articles ou votre barre de widget grâce à son plugin officiel. 



== Installation ==

Comment fonctionne le plugin Neeed ? 

** Vous devez disposez d'un compte [Neeed.com](http://neeed.com/) pour faire fonctionner le plugin Neeed **

1. Uploadez le plugin dans le répertoire '/wp-content/plugins/'
2. Activez le plugin par le menu d'administration des plugins de Wordpress
3. Rendez vous dans le menu 'Neeed' et configurez votre plugin

Retrouvez votre clef d'API dans les paramètres de votre compte [Neeed](http://neeed.com/)

Vous pouvez : 

1. Affichez vos articles dans la barre latérale de votre blog 
2. Affichez vos produits, une liste ou un profil découvert dans vos articles grâce au tag [neeed]
3. Affichez et customisez l’affichage de vos produits grâce à la fonction PHP neeed() 


== Frequently Asked Questions == 


= Où trouver ma clef d'API Neeed ? =

Dans les paramètres de [votre compte Neeed](http://neeed.com/)

= Quel est le maximum de produits affiché ? =

15 produits 

= Comment afficher une liste spécifique ? = 

Dans le shorttag neeed vous pouvez ajouter un paramètre : list=”nom de la liste” 

= Le message d’erreur “Impossible d’utiliser le cache” apparaît = 

Le dossier cache/ contenu dans le plugin semble ne pas être accessible. 
Modifiez ses autorisations afin que votre serveur puisse écrire dedans
